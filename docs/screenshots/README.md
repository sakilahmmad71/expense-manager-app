## Guidelines for Screenshots

1. **Resolution**: Use consistent resolutions (1920x1080 for desktop, iPhone 12 Pro for mobile)
2. **Content**: Use realistic sample data, avoid personal information
3. **Format**: PNG format for better quality
4. **Naming**: Use descriptive, lowercase names with hyphens
5. **Theme**: Capture in the default theme (black and white design)

## Tools

Recommended tools for taking screenshots:

- **macOS**: Command + Shift + 4
- **Windows**: Snipping Tool or Windows + Shift + S
- **Browser Dev Tools**: For mobile responsive screenshots

## Adding Screenshots

1. Take screenshots following the guidelines above
2. Add them to this directory with appropriate names
3. Update the main README.md if you add new screenshot categories
